# 创建索引

创建 lib 索引例子，number_of_shards 为分片数量，number_of_replicas 为索引的副本数，这里主要演示创建索引时可设置一些相关参数，且为非必填项。

PUT /lib
{
  "mappings" : { 
   "properties": {
     "name" : {
        "type" : "text",
        "fields" : {
          "keyword" : {
            "type" : "keyword",
            "ignore_above" : 256  // 长度超过ignore_above设定的值，不会被索引或存储
          }
        }
      }
   }
  },
  "settings": {
    "number_of_shards": 1,
    "number_of_replicas": 1
  }
}
